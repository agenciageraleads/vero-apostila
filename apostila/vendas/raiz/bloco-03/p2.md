# Bloco 03 — Canvas
## Mapa de Inversão

Para cada momento da sua venda, escreva o que você costuma dizer — e o que o cliente precisa ouvir.
Não corrija o que você disse. Escreva o que é real.

---

| Momento | O que você costuma dizer | O que o cliente precisa ouvir |
|---------|--------------------------|-------------------------------|
| No primeiro contato | | |
| Ao apresentar o produto ou serviço | | |
| Ao explicar o diferencial | | |
| Quando surge objeção de preço | | |
| No fechamento | | |

---

Olhe para a coluna da direita. Quanto do que você disse está lá?

---

"Mas o cliente precisa entender o que vai receber." → Ele precisa entender o que vai ganhar. São coisas diferentes.

"Se eu não explicar o processo, vai parecer que qualquer um faz." → O cliente não escolhe quem explica melhor. Escolhe quem resolve o que ele precisa.

"Eu preciso mostrar minha expertise." → Mostre pelo resultado que entrega, não pelo processo que usa.

---

Depois deste Canvas, o empresário para de falar sobre si e começa a falar sobre o cliente. A conversa de venda muda de tom — não porque aprendeu técnica, mas porque mudou o que coloca no centro.
