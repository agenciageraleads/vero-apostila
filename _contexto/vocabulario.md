# Vocabulario Vero — Regua Linguistica

> Todo agente DEVE consultar este arquivo antes de escrever qualquer conteudo.
> Violacao de vocabulario = bloco volta automaticamente.

---

## Palavras e Expressoes PROIBIDAS

Estas palavras/expressoes NAO podem aparecer em nenhum bloco da apostila:

### Linguagem Motivacional
- "Voce consegue"
- "Acredite em voce"
- "O ceu e o limite"
- "Basta querer"
- "Sonhe grande"
- Qualquer variacao de autoajuda ou coaching

### Promessas Vazias
- "Sucesso" (como promessa ou resultado garantido)
- "Crescer" / "Crescimento" (como promessa — pode ser usado como descricao de estado)
- "Solucao" / "A solucao e..."
- "Resultado garantido"
- "Transformacao" (como promessa magica)

### Jargao Academico
- Termos tecnicos sem contexto pratico
- Linguagem de MBA/consultoria generica
- Frameworks famosos como se fossem verdade universal

### Linguagem de Metodo (proibida na P2)
- "Canvas" — nunca usar no texto que o empresario le (e ferramenta interna, nao linguagem da apostila)
- "Depois deste Canvas..." — substituir por: "Feito isso", "Depois disso", "Ao terminar", "Com isso claro"
- Terceira pessoa no resultado esperado: "o empresario para de..." — sempre segunda pessoa: "voce para de..."

### Tom Incorreto
- Linguagem de palco / palestrante
- Ironia excessiva
- Condescendencia ("voce precisa entender que...")
- Suavizacao de desconforto para agradar

## Palavras e Expressoes PERMITIDAS / ENCORAJADAS

### Conceitos-Chave Vero
- **Controle** — estado operacional, nao microgerenciamento
- **Previsibilidade** — saber o que vai acontecer antes de acontecer
- **Improviso** — o inimigo central que a Vero combate
- **Criterio** — base de decisao sustentavel
- **Sustentacao** — manter no tempo, nao apenas iniciar
- **Renuncia** — dizer nao com criterio (especialmente Gestao)
- **Dispensabilidade** — empresa que funciona sem o dono (Legado)
- **Maturidade operacional** — estado real, nao tamanho

### Verbos que Definem a Vero
- Eliminar (improviso)
- Sustentar (controle)
- Renunciar (ao excesso)
- Comprovar (status)
- Operar (com criterio)

## Frases Institucionais Vero

Podem ser usadas como referencia de tom, nunca como slogan repetitivo:

- "Quem executa, permanece"
- "Status nao se compra. Status se comprova."
- "Formacao sem criterio gera ilusao"
- "Criterio sem execucao nao se sustenta"
- "Empresa madura e aquela que se sustenta sem heroismo"
- "A Vero nao forma empresarios no sentido tradicional. A Vero elimina improviso."

## Tom de Voz

| Deve ser | Nunca deve ser |
|----------|---------------|
| Adulto | Infantil ou paternalista |
| Direto | Enrolado ou evasivo |
| Realista | Otimista ingenuo |
| Provocativo (com criterio) | Agressivo ou arrogante |
| Observador | Julgador |
| Concreto (exemplos reais) | Abstrato (teorias soltas) |

### Regra de Ouro do Tom
O texto deve soar como alguem que **ja viu o problema acontecer muitas vezes** e esta descrevendo o que observou — nao como alguem que esta ensinando o que voce deveria fazer.
