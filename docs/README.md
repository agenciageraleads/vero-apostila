# Vero — Apostila de Formação

**Bem-vindo à escola que comprova status através da execução.**

Este é o material de formação, validação e certificação operacional da **Vero Escola de Empresários**.

---

## Estrutura

A apostila é organizada em **4 pilares** × **4 fases** × **22 blocos cada** = 352 blocos de conteúdo.

### Pilares

1. **Financeiro** ✅ (22/22 blocos) — Como sua empresa gera caixa de forma previsível
2. **Operação** ⬜ (em breve) — Como eliminar improviso através de padrões
3. **Vendas** ⬜ (em breve) — Onde e como sua empresa compete
4. **Pessoas** ⬜ (em breve) — Como criar cultura que persiste sem você

### Fases de Maturidade

| Fase | Problema Central | O que Muda |
|------|-----------------|-----------|
| 🌱 **Raiz** | Falta de previsibilidade | Empresário para de se perder |
| 🛤️ **Trilha** | Inconsistência | Empresário para de recomeçar |
| 🧠 **Gestão** | Excesso | Empresário para de acumular |
| 🏛️ **Legado** | Dependência invisível | Empresário para de ser necessário |

---

## Como Usar

### Navegação
- [Voltar para Home](/)
- Clique em cada pilar para ver as fases
- Clique em cada fase para ver os 22 blocos
- Cada bloco tem 2 páginas:
  - **Página 1 — Narrativa**: Histórias reais que você reconhece
  - **Página 2 — Canvas**: Onde você pratica

### Leitura
- Cada página é planejada para impressão (A4)
- Leia onde quiser: tela ou papel
- Preencha o canvas à mão (é obrigatório para aprendizado)

### Progresso
Não há ordem de leitura obrigatória dentro de cada pilar. Você pode pular entre blocos, mas:
- Complete os **4 pilares em sequência**: Financeiro → Operação → Vendas → Pessoas
- Dentro de cada pilar, respeite as **4 fases**: Raiz → Trilha → Gestão → Legado

---

## Princípios Vero

- **Status não se compra.** Status se comprova.
- **Formação sem critério gera ilusão.** Execução sustentada comprova status.
- **Quem executa, permanece.** Quem não executa, desaparece.
