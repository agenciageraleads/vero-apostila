---

"Mas meus clientes sabem o que eu faço." → Os que já compraram, sim. Quem está decidindo se compra, ainda não.

"Meu trabalho é complexo demais para simplificar." → Complexidade é interna. O cliente só precisa entender o que muda para ele.

"Tenho explicado assim há anos." → Quantas vendas você perdeu sem saber disso?

---

Depois deste Canvas, o empresário para de explicar processo e começa a nomear resultado. A oferta fica reconhecível para quem ainda não comprou. Se o Passo 05 revelou que a frase ainda confunde, o problema não é o cliente — é a mensagem.
