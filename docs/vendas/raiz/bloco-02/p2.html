03
O que o cliente ganha de concreto
Não o que você faz. O que muda na vida ou no negócio dele depois que contratar você.
↓
04
Reescreva em uma frase
Use só as palavras do passo anterior. Uma frase. Ponto de vista de quem compra.
↓
05
Teste com alguém de fora do seu setor
Mostre a frase do passo 04. Escreva o que essa pessoa entendeu — não o que você quis dizer.
"Mas meus clientes sabem o que eu faço." → Os que já compraram, sim. Quem está decidindo se compra, ainda não.
"Meu trabalho é complexo demais para simplificar." → Complexidade é interna. O cliente só precisa entender o que muda para ele.
"Tenho explicado assim há anos." → Quantas vendas você perdeu sem saber disso?
Depois deste Canvas, o empresário para de explicar processo e começa a nomear resultado. Se o Passo 05 revelou que a frase ainda confunde, o problema não é o cliente — é a mensagem. Se dói, funcionou.
