Critério a partir de agora — quem entra e quem sai:
"Não posso ser seletivo, preciso de venda." → Você já está sendo seletivo. Só que está escolhendo os piores.
"Cada cliente é diferente." → Sim. Por isso você precisa de critério para separar os que funcionam dos que drenam.
"E se eu errar na triagem?" → Vai errar. Mas vai errar menos do que errando sem nenhum critério.
Depois deste Canvas, o empresário para de aceitar qualquer cliente por medo e começa a escolher com critério. O filtro existe antes da venda, não depois do problema. Se dói, funcionou.
