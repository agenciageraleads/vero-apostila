"Mas eu pago os impostos em dia."

Pagar em dia não é provisionar. Se você não separou antes de gastar, pagou com o dinheiro de outra coisa.

"Equipamento quebrado é imprevisto, não é custo."

Imprevisto que se repete é custo. Você só não colocou no preço.

"Meu pró-labore é o que sobra."

Então você não tem pró-labore. Tem sobra. São coisas diferentes.

Feito isso, você para de chamar de lucro o que ainda vai ser cobrado. O número vai ser menor. Isso não é problema — é a primeira vez que ele é verdadeiro.
