**Dúvidas comuns:**

— *"Mas cada gasto tinha uma razão. Nada foi por impulso."*
Justificativa não é critério. Qualquer gasto tem uma razão — o que falta é saber qual gasto vem antes dos outros.

— *"Eu não posso controlar o que aparece no meio do mês."*
Não é sobre controlar o que aparece. É sobre saber o que você escolhe quando o inesperado aparece.

— *"Eu ia cortar, mas era uma oportunidade boa demais para perder."*
Oportunidade sem parâmetro é qualquer gasto com uma boa história. O caixa não distingue os dois.

---

**Resultado esperado:**
Se você não soube responder à última pergunta, o problema não é o gasto — é a ausência de parâmetro para decidir. Enquanto isso não muda, qualquer mês pode repetir o de Sandro.
